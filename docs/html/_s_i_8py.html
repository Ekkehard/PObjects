Detailed Description

Version
2.0.0
Purpose
Implement the International System of Units (SI Units) for Physics toolbox package for Python.
Comments
This code replaces the module SIPrefix from the same author. It combines the functions from this module into a Prefix class and adds a Unit class. The code in this module is the "secret sauce" behind the PObject class.
Since this code uses "μ" to represent the e-06 prefix and "Ω" to represent Ohms, this file must be utf-8 encoded. In input strings, both "μ" and "u" as well as "Ohm" and "Ω" are accepted. In output strings "μ" and "Ω" are used unless the parameter strictAscii is set to True.
