Detailed Description

Convenience class for EEObject initialized as inductor.

Since it is derived from EEObject and not PObject, objects will be confined to a series that must be supplied when initializing it - there is no default series for inductors. To create an inductor object with values not bound to a particular series, use a PObject with unit "H".
